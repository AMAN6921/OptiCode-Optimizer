# Owl AI

Engineering education platform built with React and TypeScript.

## Tech Stack

- React 18.3.1
- TypeScript 5.8.3
- Vite 5.4.19
- Tailwind CSS 3.4.17
- shadcn/ui

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run lint` - Lint code
- `npm run test` - Run tests
